CSS Cheat Sheet

1. border

'border' define un atributo a todos los bordes de los elementos seleccionados.\n Con 'border' se puede definir valores como el color o el ancho.

ELEMENTO

2. border-left, border-right, border-top, border-bottom

'border-...' son propiedades rapidas para definir el ancho, estilo y color de los bordes respectivos de un elemento.

ELEMENTO

4. background-image

5. color

'color' es una propiedad que define el valor de color para un elemento dado. Este valor puede ser en formato de texto (nombre del color), RGB, o HEX.

ELEMENTO

6. font

'font' permite establecer valores para todas las propiedades relacionadas con la tipografia: font-style, font-variant, etc.

ELEMENTO

7. font-family

'font-family' define una lista de fuentes con un orden de prioridad para utilizar en un elemento dado. Los valores se separan con comas (",").

ELEMENTO

8. font-style, font-weight

'font-style' y 'font-weight' son propiedades que definen atributos para una fuente. 'font-style' permite definir de entre los aspectos 'normal', 'italic' y 'oblique'.

ELEMENTO

9. width, height

'width' y 'height' especifican el ancho y alto del area para un elemento respectivamente. Esta area se ecuentra dentro del padding, borde y margen de dicho elemento.

ELEMENTO

10. list-style

'list-style' permite definir todos los parametros para uba lista dada.

11. margin

'margin' establece el margen para los 4 lados. Es una abreviacion para evitar establecer cada margen por separado.

ELEMENTO

12. padding

'padding' establece el espacio de relleno requerido por todos los lados de un elemento. Es referente al espacio entre un elemento y su borde.

ELEMENTO

13. overflow

'overflow' especifica la forma en que se representa un elemento: ya sea recortar el contenido, dibujar barras de desplazamiento, o mostrar el contenido excedente en un elemento a nivel de bloque.

14. display

'display' especifica si un elemento se maneja como block o "inline element" y el formato usado por sus hijos. Establece los tipos de visualizacion interna y externa de un elemento.

ELEMENTO

15. position

'position' especifica la posicion de un elemento en el documento. La posicion final se determina mediante 'top', 'right', 'bottom', y 'left'.

ELEMENTO

16. filter

'filter' da efectos graficos como el desenfoque o el cambio de color a un elemento representado. Se usan comunmente para representacion de imagenes, fondos, y bordes.
